## О проекте

**laba12_Toriev_Sali** — это консольный генератор надёжных паролей, написанный на `Python`. Проект демонстрирует правильное оформление технической документации в формате **Markdown**.

Основная цель — предоставить простой, но гибкий инструмент для создания паролей, устойчивых к атакам перебора. Программа поддерживает настройку длины и состава символов.

> «Слабый пароль — это приглашение для злоумышленника. Используй генератор, чтобы не рисковать.»  
> *— из методических рекомендаций кафедры ДЦРСиЭ* [^1]

---

## Возможности

- ✅ Генерация паролей длиной от `6` до `64` символов.
- ✅ Выбор типов символов:
  - строчные буквы `a-z`
  - заглавные буквы `A-Z`
  - цифры `0-9`
  - спецсимволы `!@#$%^&*()`
- ✅ Исключение похожих символов (`1`, `l`, `O`, `0`).
- ✅ Простота использования — запуск одной командой.
