## О проекте
Консольный генератор паролей на **Python**.  
**Цель:** создание надёжных паролей с настраиваемыми параметрами.

> **Миссия проекта:** *«Безопасность не должна быть сложной»* [^1].

## Возможности
- Генерация паролей длиной **6–64** символа.
- Выбор типов символов:
  - строчные буквы (`a-z`)
  - заглавные буквы (`A-Z`)
  - цифры (`0-9`)
  - спецсимволы (`!@#$%^&*()`)
- Исключение похожих символов (`1`, `l`, `O`, `0`).
- Простой запуск из командной строки.

## Установка
1. Убедитесь, что установлен Python 3.8+:  
   `python --version`
2. Склонируйте репозиторий:  
   `git clone https://github.com/TorievSali/markdown-practice.git`
3. Перейдите в папку проекта:  
   `cd laba12_Toriev_Sali`
4. Запустите генератор:  
   `python password_generator.py`

## Пример использования
```python
import random
import string

def generate_password(length=12):
    chars = string.ascii_letters + string.digits + "!@#$%"
    return ''.join(random.choice(chars) for _ in range(length))

print(generate_password(16))
