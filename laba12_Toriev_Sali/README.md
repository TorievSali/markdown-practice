## О проекте
Консольный генератор паролей на **Python**. Цель — создание надёжных паролей.

> Миссия проекта: безопасность через простоту. [^1]

## Возможности
- Длина 6–64 символов
- Выбор символов:
  * буквы (a-z, A-Z)
  * цифры
  * спецсимволы
- Исключение похожих (`1`, `l`, `O`, `0`)

## Установка
1. Проверьте Python: `python --version`
2. Склонируйте репозиторий: `git clone <url>`
3. Запустите: `python password_generator.py`
