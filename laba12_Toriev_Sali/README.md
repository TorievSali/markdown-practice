# 📖 О проекте

**TaskFlow** — это *консольное приложение* для управления задачами.

Программа позволяет:

- создавать задачи;
- удалять задачи;
- отмечать выполненные задания;
- сохранять список дел.

> Хороший план сегодня лучше идеального плана завтра.

Проект написан на языке `Python`.

# ❓ Контрольные вопросы

## 1. Чем отличается вложенная цитата от обычной?

Вложенная цитата использует несколько символов `>>`,
а обычная — один символ `>`.

## 2. Как сделать сноску в Markdown и где она отображается?

Сноска создаётся с помощью конструкции:

```md
[^1]
```

Описание сноски отображается внизу документа.

## 3. В чём разница между - [ ] и - [x] в списках задач?

- `[ ]` — задача не выполнена;
- `[x]` — задача выполнена.

## 4. Какие два способа вставки эмодзи вы знаете?

1. Через код:
```md
:rocket:
```

## 5. Зачем нужны HTML-вставки? Приведите пример.

HTML-вставки расширяют возможности Markdown.

Пример:

```html
<details>
<summary>Текст</summary>
</details>
```

## 6. Почему важно использовать правильные имена для внутренних ссылок (якорей)?

Чтобы ссылки правильно переходили к нужному разделу документа.

## 7. Что произойдёт, если в таблице окажется разное количество столбцов в разных строках?

Таблица может отображаться некорректно.
