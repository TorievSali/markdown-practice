# Блок-схема алгоритма решения квадратного уравнения

**Описание алгоритма:**  
Пользователь вводит коэффициенты a, b, c квадратного уравнения ax² + bx + c = 0.  
Вычисляется дискриминант. В зависимости от его значения определяются и выводятся корни.

## Диаграмма

```mermaid
flowchart TD

    A([Начало]) --> B[/Ввести a, b, c/]

    B --> C{a = 0?}

    C -- Да --> D[Вывести сообщение об ошибке]
    D --> E([Конец])

    C -- Нет --> F[Найти дискриминант D]

    F --> G{D < 0?}

    G -- Да --> H[Вывести "Корней нет"]
    H --> E

    G -- Нет --> I{D = 0?}

    I -- Да --> J[Вычислить x]
    J --> K[Вывести корень]
    K --> E

    I -- Нет --> L[Вычислить x1 и x2]
    L --> M[Вывести два корня]
    M --> E
```
