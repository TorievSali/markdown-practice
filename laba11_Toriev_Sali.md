# Линейные алгоритмы: понятие и примеры

## 1. Определение линейного алгоритма

**Линейный алгоритм** — это алгоритм, в котором команды выполняются
последовательно сверху вниз без циклов и условий.

Особенности линейного алгоритма:

- простая структура;
- команды выполняются по порядку;
- отсутствуют ветвления;
- отсутствуют циклы.

> Алгоритм – это точно определённая последовательность действий.

---

## 2. Примеры линейных алгоритмов

### 2.1. Вычисление арифметического выражения

Задача: вычислить значение выражения.

Формула:

`y = (a + b) * 2`

Пример программы на Python:

```python
a = 5
b = 3

y = (a + b) * 2

print("Результат:", y)
```

Пояснение:

- переменные `a` и `b` хранят числа;
- вычисляется арифметическое выражение;
- результат выводится на экран.

---

### 2.2. Обмен значениями двух переменных

Задача: поменять местами значения двух переменных.

Пример программы:

```python
a = 10
b = 20

print("До обмена:")
print("a =", a)
print("b =", b)

temp = a
a = b
b = temp

print("После обмена:")
print("a =", a)
print("b =", b)
```

Пояснение к коду:

1. Значение переменной `a` сохраняется во временную переменную `temp`;
2. Значение `b` записывается в `a`;
3. Значение `temp` записывается в `b`.

---

## 3. Особенности реализации на языках программирования

Особенности линейных алгоритмов:

- простота написания;
- высокая читаемость кода;
- быстрое выполнение;
- используются практически во всех программах.

# Контрольные вопросы

## 1. Какие расширения имеют файлы Markdown?

Файлы Markdown имеют расширения `.md` и `.markdown`.

## 2. Как создать заголовок второго уровня?

Нужно поставить два символа `##` перед текстом заголовка.

Пример:

```md
## Заголовок
```

## 3. Чем отличается маркированный список от нумерованного?

Маркированный список использует символы `-`, `*`, `+`,
а нумерованный список использует цифры.

## 4. Как вставить изображение в документ?

Изображение вставляется с помощью конструкции:

```md
![Описание](ссылка_или_путь)
```

## 5. Для чего нужны блоки кода и как указать язык программирования?

Блоки кода нужны для оформления программного кода
с сохранением форматирования и подсветки синтаксиса.
